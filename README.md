## ⚙️ Example Profile Configuration

To tailor Origami for a specific workflow, create a `.origami_profile.json` file in your user directory:

```json
{
  "version": "3.4.4",
  "user": {
    "name": "Jane Doe",
    "preferred_language": "en-US",
    "theme": "midnight"
  },
  "processing": {
    "default_output_format": "PDF/A-3",
    "compression_level": "high",
    "enable_ocr": true,
    "auto_rotate": true
  },
  "ai_integration": {
    "openai_api_key": "sk-your--here",
    "claude_api_key": "sk-ant-your--here",
    "auto_summarize_on_open": false,
    "translation_service": "openai"
  },
  "cloud": {
    "default_provider": "dropbox",
    "auto_sync": true,
    "interval_minutes": 15
  },
  "ui": {
    "toolbar_style": "icons_and_text",
    "preview_mode": "continuous_scroll",
    "zoom_step": 10
  }
}
```

This configuration activates AI summarization, high-compression exports, and cloud auto-sync—a setup ideal for busy legal professionals or creative studios.

## 🖥️ Example Console Invocation

Run Origami from the command line for  workflows:

```bash
origami-cli --input "/documents/report.pdf" \
            --output "/archive/report_final.pdf" \
            --merge "/appendices/appendix_a.pdf,/appendices/appendix_b.pdf" \
            --encrypt --password="S3cur3P@ss" \
            --add-watermark "CONFIDENTIAL" \
            --ai-summarize --summary-lang en \
            --cloud-upload dropbox:/Projects
```

This command merges a main report with two appendices, encrypts the result, adds a watermark, generates an AI summary via the OpenAI API, and uploads the final file to Dropbox—all in one seamless fold.

## 🧠 AI Integration: OpenAI & Claude API

Origami 3.4.4 is the first document tool to offer **dual AI integration**. Here’s how they complement each other:

- **OpenAI API** – Best for general-purpose tasks: summarization, translation, text generation, and Q&A over document content. Use it to turn a 100-page research PDF into a one-paragraph digest.
- **Claude API** – Excels at nuanced analysis: legal contract review, scientific paper interpretation, and multi-step reasoning. Claude can identify clauses, detect contradictions, or explain complex concepts in plain language.

Both APIs are accessed via the **AI Panel** in the app or through the console. Your API  remain encrypted locally and are never transmitted to third parties.
